# 🎰 Lottery React Game

A fun and simple lottery game built using **ReactJS**! 

## 🛠️ Installation

git clone https://github.com/gauravag18/Lottery-React-Game.git
cd Lottery-React-Game
npm install
npm run dev


## 📂 Project Structure

```
Lottery-React-Game/
├── public/
├── src/
│   ├── components/       # Reusable components (if any)
│   ├── App.js            # Main application logic

```
